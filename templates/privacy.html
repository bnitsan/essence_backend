Privacy Policy



  • We do not pass or sell any data to third parties.
  • We only inspect non-private queries which have been flagged as good or bad.
  • Only pages requested for Processing are sent to the server.
  • The app does not access any information beyond the current page.
  • We re-use cached queries only with public pages with no personal information, i.e. ones which were accessed directly by the server, not from the user-side.
  • Passwords are not stored.
  • The Notion integration only has access to write new database entries, not to read or update existing ones. It also does not access any user information.
